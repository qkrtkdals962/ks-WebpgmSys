# software폴더에 하위폴더를 만들고 tree로 표현하기

### 사용한 명령어
- **mkdir**
  - 디렉토리(폴더) 생성
- **cd 디렉토리이름, ..**
  - 해당 디렉토리로 이동
  - 부모 디렉토리로 이동
- **cat**
  - **cat >**을 이용하여 새로운 txt를 만들고 바로 Ctrl+D를 사용하여 빈 txt를 생성
- **tree**
  - **tree**명령어를 사용하여 tree구조 표현
